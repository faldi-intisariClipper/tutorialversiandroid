Tahap 2: Instalasi
Sekali Klik

  • 1

    Buka aplikasi Termux yang baru saja Anda instal. (Tunggu beberapa detik saat layar hitam hingga muncul tulisan "Welcome to Termux!").

  • 2

    Salin (Copy) kode script instalasi di layar sebelah / di bawah ini.

  • 3

    Tempel (Paste) ke dalam Terminal lalu tekan tombol Enter di keyboard HP Anda. Biarkan sistem mengekstraksi file otomatis.

pkg update -y && pkg install wget -y && wget https://github.com/intisariapps-com/Intisari-AutoCut/raw/main/intisari-autocut_latest.deb -O intisari-autocut_latest.deb && pkg install ./intisari-autocut_latest.deb -y && rm -f intisari-autocut_latest.deb && intisari
